**Sales Performance Analysis Dashboard (Excel Project)**

This project is inspired by a dashboard demo created by **Daniel on YouTube**. I recreated the project from scratch and added my own improvements to better understand how sales performance varies across **products**, **regions**, and **customer segments**.

Using **Excel** for both **data cleaning** and **visualization**, I built an interactive dashboard that uncovers key business insights and highlights trends that can support better decision-making.
